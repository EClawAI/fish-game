# 🐟 大鱼吃小鱼 (Fish Eat Fish)

经典的大鱼吃小鱼游戏，使用 HTML5 Canvas 开发。

## 🎮 游戏说明

控制你的鱼，在海洋中生存！

- **吃掉比你小的鱼** - 获得分数并长大
- **躲避比你大的鱼** - 否则会被吃掉
- **生存更长时间** - 挑战高分

## 🕹️ 操作方式

- **鼠标**: 移动鼠标控制鱼的游动方向
- **触摸**: 在触摸屏上滑动手指控制

## 🚀 快速开始

### 方法 1: 直接打开
```bash
# 直接在浏览器打开 index.html
open index.html
# 或
xdg-open index.html
```

### 方法 2: 使用本地服务器
```bash
# Python 3
python3 -m http.server 8000

# 然后访问 http://localhost:8000
```

### 方法 3: 使用 Node.js
```bash
npx serve .
```

## 🎯 游戏特性

- ✨ 流畅的 60FPS 动画
- 🎨 精美的鱼类设计和动画
- 🌊 动态海洋背景（气泡、海草）
- 📈 难度递增（生成速度随分数提高）
- 📱 支持鼠标和触摸控制
- 🎪 粒子爆炸效果

## 🏆 游戏技巧

1. **初期**: 快速吃掉小鱼，尽快长大
2. **中期**: 谨慎选择目标，避免被大鱼包围
3. **后期**: 利用速度优势，灵活躲避

## 💻 技术栈

- **HTML5 Canvas** - 游戏渲染
- **JavaScript ES6+** - 游戏逻辑
- **CSS3** - UI 样式

## 📊 游戏数据

| 属性 | 说明 |
|------|------|
| 初始大小 | 15 |
| 最大大小 | 80 |
| 敌人速度 | 1-3 |
| 玩家速度 | 2-5 (随大小变化) |

## 🎨 鱼类颜色

游戏包含 12 种随机颜色的鱼：
- 红色、青色、蓝色、绿色
- 黄色、紫色、粉色等

## 📝 开发说明

### 核心类

- **Fish** - 鱼类（玩家和敌人）
- **Particle** - 粒子效果
- **Bubble** - 背景气泡
- **Game** - 游戏主控制器

### 碰撞检测

```javascript
// 简单圆形碰撞
distance < playerSize + enemySize
```

### 成长系统

```javascript
// 吃掉小鱼后成长
player.grow(enemy.size * 0.05);
```
